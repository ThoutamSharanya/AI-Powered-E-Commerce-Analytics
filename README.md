\# AI-Powered E-Commerce Analytics System



\## Project Overview



This project uses Machine Learning and Data Analytics to improve e-commerce decision-making.



\## Features



\* Customer Churn Prediction

\* Delivery Delay Prediction

\* Customer Segmentation

\* Sales Forecasting

\* Interactive Streamlit Dashboard



\## Technologies Used



\* Python

\* Pandas

\* NumPy

\* Scikit-Learn

\* Streamlit

\* GitHub



\## Models Used



\* Random Forest Classifier

\* K-Means Clustering

\* Linear Regression



\## Dataset



Brazilian E-Commerce Public Dataset (Olist)



\## Project Structure



\* app.py

\* delay\_model.pkl

\* customer\_segmentation.pkl

\* sales\_forecast.pkl



\## How to Run



streamlit run app.py
